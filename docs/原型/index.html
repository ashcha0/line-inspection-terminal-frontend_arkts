🚛 AGV智能巡检系统

低保真原型页面导航

地铁隧道巡线车智能巡检系统的交互原型展示。本系统实现了AGV自动巡检、实时监控、故障识别、数据上传等核心功能。

🔧

系统初始化页面

系统启动时的自检页面，检查文件完整性、数据库连接、AGV通信和摄像头状态，确保系统正常运行。

主要功能：

系统自检 状态展示 错误诊断 设置入口
查看原型
📋

任务列表页面

巡检任务管理中心，支持任务的创建、编辑、删除、搜索和状态管理，提供完整的任务生命周期管理。

主要功能：

任务搜索 新增任务 状态管理 数据上传
查看原型
📊

任务详情页面

已完成任务的详细查看页面，展示故障图片、位置标记、处理结果，支持故障确认和备注功能。

主要功能：

故障图片 位置标记 故障确认 进度展示
查看原型
🎥

任务执行页面

实时巡检控制台，显示视频流、AGV状态、实时位置，支持车辆控制和故障实时处理。

主要功能：

实时视频 车辆控制 实时监控 故障处理
查看原型
⚙️

系统设置页面

系统配置管理页面，设置AGV连接参数、摄像头配置、云端平台地址等系统核心参数。

主要功能：

AGV配置 摄像头设置 云端配置 参数验证
查看原型

🔄 页面流程

系统初始化 → 任务列表 → 新增/执行任务 → 任务详情 ↗ 系统设置

🛠 技术栈信息

🖼 前端框架：Vue.js 3.x
🎨 UI组件：Element Plus
🛣 路由管理：Vue Router
📦 状态管理：Pinia
🔌 HTTP客户端：Axios
📱 响应式设计：CSS3 Grid/Flexbox
